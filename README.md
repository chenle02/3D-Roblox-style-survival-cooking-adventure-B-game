# Servent: Signal Lost - RoboChef Protocol

3D Roblox-style first-person survival cooking adventure prototype.

## Features

- First-person 3D-style controls
- Remote radio station survival setting
- Cleaner glass-style HUD with battery, food, parts, and signal stats
- Crosshair interaction system
- Touch joystick, WASD movement, screen-drag camera look, and arrow-key look
- RoboChef battery system
- Cooking, solar recharge, generators, puzzles
- Strange transmissions and story choices

Build the APK with GitHub Actions.
